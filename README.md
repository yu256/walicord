# Walicord

Discord上で共有経費を追跡し、グループメンバー間の最適な債務決済を計算するDiscordボットです。

## ドキュメント

- **[セットアップガイド](docs/setup.md)**: インストール、環境設定、ボットの作成方法
- **[ユーザーガイド](docs/usage.md)**: 経費の入力方法、清算コマンドの使い方
- **[数学的仕様](docs/design/settlement-mathematics.md)**: 割り勘計算のアルゴリズムと端数処理
- **[トランザクション構築](docs/design/transaction-construction.md)**: 決済時の送金生成ロジック

## 機能概要

- **経費記録**: `1000 to @Hanako` のように自然な入力で記録できます。
- **最適化**: 複数の貸し借りを相殺し、最小限の送金で清算できるよう計算します。
- **現金配慮オプション**: `cash` 指定は送金構築にのみ影響し、残高計算（割り勘数学）自体は変えません。
- **残高管理**: 誰が誰にいくら貸しているかを常に追跡します。
- **画像出力**: 分かりやすい残高表を画像で生成します。

## 開発者向け

このプロジェクトは Rust で書かれています。

- `walicord`: メインアプリケーション
- `walicord-domain`: コアロジック
- `walicord-transfer-construction`: 送金構築エンジン

詳しくは `docs/` ディレクトリを参照してください。
